# Math Magicien

> The Math magicien website is a single page app that allows users to :
        > Make simple calculations
        >Read a random math-related quote


## Built With

- HTML
- CSS
- JavaScript / ES6
- React


### Prerequisites
- browser (e.g. Google Chrome)
- text editor (e.g. VSCode)
- terminal/CLI
